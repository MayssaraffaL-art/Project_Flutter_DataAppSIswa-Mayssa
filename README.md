# Simle Data App Siswa (Mobile CRUD App)

**Data App Siswa** is a simple mobile application built using **Flutter**, connected to a **PHP API** and **MySQL database**.  
This application is designed to manage student data digitally using a **CRUD (Create, Read, Update, Delete)** system.

The app features a clean interface, is easy to use, and is suitable for learning how to integrate a mobile application with a backend system.
